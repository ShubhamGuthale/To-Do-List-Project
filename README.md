
**# To Do List Application** 
 Todo application developed using JSP, Servlet, JDBC and MySQL Database.


## Following features :-
- Add todo
- Edit todo
- List Todo
- Delete Todo
- Update TODO
- Task Status


## Technology :-
- Java 8+
- Apache Tomcat server 9.0
- JSP and Servlet
- JDBC
- MySQL database
- Eclipse IDE
