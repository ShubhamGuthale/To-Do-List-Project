# Task Tracker â€“ To-Do List Management Application 
A web-based To-Do List Application built using **JSP**, **Servlets**, **JDBC**, and **MySQL**, designed to help users efficiently manage their daily tasks with full CRUD functionality and real-time status tracking.



## Following features 
- Add Todo Items
- Edit Existing Tasks
- View All Tasks (List)
- Delete Tasks
- Update Task Status (Pending/Completed)
- Task Dashboard for quick status overview


## Technology 
- Java JDK 20
- Apache Tomcat server 9.0
- JSP and Servlet (MVC architecture)
- JDBC for database connectivity
- MySQL for persistent storage
- Eclipse IDE
